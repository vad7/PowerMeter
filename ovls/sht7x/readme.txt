sht7x.ovl

Драйвер для работы с датчиками влажности по I2C типа: FOST02, SHT1x, SHT7x.
Данные с датчика усредняются за два замера в 0.54 секунды (один замер 0.27 сек). 
Всё делается по прерываниям и готовности датчика.
http://esp8266.ru/forum/attachments/sht71-gif.1549/


Инициализация:
mdb[60] - Номер пина (0..15) для сигнала SCL, по умолчанию GPIO4
mdb[61] - Номер пина (0..15) для сигнала SDA, по умолчанию GPIO5
Переменные:
mdb[62] - Счетчик опроса датчика. При смене указывает, что с датчика обработаны новые данные. 
mdb[63] - Флаг: 
			=0 - данные верны, ошибок нет. 
			=1 - драйвер запущен, ждет первого замера.
			=-1 - драйвер закрыт.
			=-2 - неисправность сенсора влажности
			=-3 - ошибки в параметрах инициализации (номера пинов SCL, SDA)
mdb[64] - Температура в 0.01 градуса (со знаком)
mdb[65] - Влажность в 0.01%
